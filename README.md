# Claude Certified Architect — Foundations · Video Series

A free, open-source video series by [GenAI Mentor](https://www.genai-mentor.ai/) that takes you from zero to passing the **Claude Certified Architect — Foundations** exam.

Every module maps directly to the official exam content outline. Every video ends with a hands-on lab you can commit to your own portfolio.

> 📘 Official exam guide: [claudecertifications.com/claude-certified-architect/exam-guide](https://claudecertifications.com/claude-certified-architect/exam-guide)

## Who this is for

You have **6+ months of hands-on Claude experience** — you've written API calls, used Claude Code, built at least one agent — and you want to pass the foundations exam on the first try. If you're brand new to Claude, start with the [Build AI](https://www.genai-mentor.ai/build-ai/) path first.

## Start here

1. Watch **[Module 0](./module-00-orientation/)** (20 min) to set expectations and set up your lab.
2. Start recording / studying with **[Module 1 · Video 1 — The Agentic Loop](./module-01-agentic-architecture/video-01-agentic-loop/)**.
3. Before the exam, sit the **[60-question mock](./module-07-exam-strategy-mock/video-02-mock-exam/mock-exam.md)**.
